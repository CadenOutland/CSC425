# creating website using copilot
This project is a small website built using GitHub Copilot. It includes four static pages and one dynamic page created with JavaScript.

The website serves as a basic personal or business site.

# files
A homepage (index.html) with navigation buttons

Three static info pages:
 About:    About page(about.html)
 Contact:  contact page(contact.html)
 Register: Registrartion page(registration.html)

One interactive page (dynamic.html) where users can enter text and see a dynamic list update in real time using JavaScript

css/styles.css — Styles for all pages
js/script.js — JavaScript for interactive list

## Workflow

- Created branches for adding pages and dynamic features
- Used pull requests to merge features into the main branch
- Pushed all files to a public GitHub repository
