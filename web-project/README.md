# Web Project

This project includes a static homepage, three linked pages (About, Contact, Register), and an interactive page that allows users to add items to a list.

## Project Structure

- `index.html`: Homepage with navigation links.
- `about.html`: About page with information about the project.
- `contact.html`: Contact page with contact information.
- `register.html`: Registration page.
- `dynamic.html`: Interactive page to add items to a list.
- `css/styles.css`: Styles for the project.
- `js/script.js`: JavaScript for the interactive page.
